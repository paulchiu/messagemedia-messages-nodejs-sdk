# MessageMedia Messages NodeJS SDK Patched Fork

> [!IMPORTANT]
> The original MessageMedia SDK is no longer maintained upstream.

This repository is a fork of `messagemedia/messages-nodejs-sdk` ([GitHub](https://github.com/messagemedia/messages-nodejs-sdk)). The upstream SDK has not been actively maintained for several years. This fork exists to keep the package keep vulnerability fixes up to date.

If you're building new applications against the Sinch MessageMedia Messaging API, they recommend using direct REST API, please see  
[MessageMedia Messaging REST API Docs](https://messagemedia.github.io/documentation/#tag/Messages).
